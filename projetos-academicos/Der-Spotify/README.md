# DER Spotify

Projeto de modelagem de banco de dados inspirado na plataforma Spotify, desenvolvido para a disciplina de Banco de Dados.

## Objetivo
Representar entidades, relacionamentos e cardinalidades de uma plataforma de streaming musical utilizando o modelo Entidade-Relacionamento (DER).

## Funcionalidades Modeladas
- Contas e perfis
- Playlists
- Músicas
- Podcasts e episódios
- Biblioteca
- Histórico
- Categorias e gêneros
- Dispositivos
- Artistas

## Tecnologias Utilizadas
- brModelo
- Modelagem DER

## Arquivos
- Modelo editável (.brM3)
- Imagem do DER (.png)
